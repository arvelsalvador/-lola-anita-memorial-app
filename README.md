# Lola Anita Memorial App 🌹

A Flutter mobile application built in loving memory of **Anita Daiz Lumbao** (1940 – 2025) — beloved grandmother, keeper of stories.

## About

This app is a digital memorial dedicated to Lola Anita. It brings together cherished memories, photos, stories, and tributes from family and loved ones in one heartfelt place.

## Features

- **Home** – A welcoming overview celebrating Lola Anita's life
- **Gallery** – A photo gallery of treasured moments
- **Memories** – Personal memories shared by family and friends
- **Story** – The story of her life
- **Tributes** – Messages and tributes from those who loved her
- **Favorites** – Favourite moments and quotes

## Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (3.x or later)
- Android Studio / Xcode for mobile builds

### Running the App

```bash
flutter pub get
flutter run
```

### Building for Production

```bash
# Android
flutter build apk

# iOS
flutter build ios
```

## Tech Stack

- **Framework:** Flutter / Dart
- **Architecture:** MVC (Model-View-Controller)
- **Platforms:** Android, iOS, Web, macOS, Linux, Windows
